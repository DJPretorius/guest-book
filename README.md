## Guest Book

This is a simple demo of Laravel 9 with VueJs and Inertia JS to create a simple guest book program.

## Setup

Run the following commands to setup the service locally. We use `docker` for setup and `docker-compose` for creating and starting our DB container.
```
docker run --rm -v ${PWD}:/app composer install
npm install
docker-compose -f .docker/docker-local.yml up -d
php artisan migrate
php artisan db:seed 
```
